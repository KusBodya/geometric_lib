


# Данная библиотека позволяет находить площади и периметры 4 фигур: круга,квадрата,треугольника,прямоугольника.


# square

## area(a)

```
       Функция "площадь квадрата"
       На вход : число (а) (длина стророны квадрата).
       (a)  является стороной квадрата
       Функция возвращает a*a,что яляется площадью квадрата. 

       >>> area(5)
       25
       
       >>> area(0.5)
       0.25
```

## perimeter(a)

```
       Функция "периметр квадрата"
       На вход : число (а) (длина стророны квадрата).
       (a) является стороной квадрата
       Функция возвращает a*4,что яляется периметром прямоугольника .

       >>>perimeter(2)
       8

       >>>perimeter(0.25)
       1
```

# circle

## area(r)

```
       Функция "площадь круга"
       На вход: число (r) (радиус круга)
       Функция возвращает pi*r*r,(pi математическая константа),что является площадью круга

       >>>area(3)
       28.274

       >>>area(0.5)
       0.785 
```

## perimeter(r)

```
        Функция "периметр окружности"
        На вход: число (r) (радиус круга)
        Функция возвращает 2*pi*r,(pi математическая константа),что является периметром окружности

        >>>perimeter(2)
        12.566

        >>>perimeter(0.3)
        1.884
```

# rectangle

## area(a,b)

```
        Функция "площадь прямоугольника"
        На вход : число (а), число (b) (длины стророн прямоугольника) .
        (a) и (b) являются шириной, либо длиной прямоугольника
        Функция возвращает a*b,что яляется площадью прямоугольника.

        >>>area(2,3)
        6

        >>>area(0.5,1.5)
        0.75
```

## perimeter(a,b)

```
        Функция "периметр прямоугольника"
        На вход : число (а), число (b) (длины стророн прямоугольника).
        (a) и (b) являются шириной, либо длиной прямоугольника
        Функция возвращает 2*(a+b),что яляется периметром прямоугольника.

        >>>perimeter(2,4)
        12
  
        >>>perimeter(0.3,0.8)
        2.2
```

# triangle

## area(a,h)

```
      Функция "площадь треугольника"
      На вход : число (а),(h) (длина стророны и высоты квадрата) .
      (a)  является стороной квадрата
      (h) является высотой,проведенно к стороне (а)
      Функция возвращает a*h*(0.5),что яляется площадью треугольника .

      >>>area(4,3)
      6

      >>>area(0.5,0.8)
      0.2
```

## perimeter(a,b,c)

```
      Функция "периметр  треугольника"
      На вход : число (а),(b),(c) (длина стророн треугольника) .
      a),(b),(c)  являются сторонами квадрата
      Функция возвращает a+b+c,что яляется периметром треугольника .

      >>>perimeter(2,3,4)
      9
 
      >>>perimeter(0.5,1.15,1,2)
      2.85
```
